これは、GrxUIの設定ファイルに関するドキュメントです。
eucjis形式でコミットして下さい。

project(grxuirc.xml)
*
	*mode(,)
		*item
	
		*view

project(robot.xml)
*
	*mode "HRP3 Control"
		*item
		*view("com.generalrobotix.ui.robot.GrxIOBClientView", "IOBClient")
			"loadSeqplay": seqplayプラグインが必要ない場合はfalseを指定する。
		
		*view("com.generalrobotix.ui.view.GrxLoggerView", "Logger")
		  "timeFormat" : String : "%8.3f" : slider横に表示する時刻のフォーマット。
	    "showFrameRateSlider" : boolean : false : frameRate設定スライダを表示
	    
	  *view("com.generalrobotix.ui.view.GrxJythonInterpreter", "Python Interpreter")
	    "maxRowCount" : int : 500 : interpreterエリアの最大行数
